PPD – Personal Productivity Dashboard

A clean, modern **personal productivity dashboard** to help you manage tasks, boost focus, track habits, and stay organized. Built as a self-taught full-stack project with a focus on simple, effective UI and functionality.

Live Demo: [ppd-iota.vercel.app](https://ppd-brown.vercel.app/)



## ✨ Features
- Task management (add, edit, delete, mark complete)
- Habit tracking with streaks/calendar view
- Focus mode / Pomodoro-style timer
- Responsive design for desktop & mobile
- Clean, minimal UI

## 🛠️ Tech Stack
- **Frontend**: JavaScript (core), HTML5, CSS
- **Possible enhancements**: React.js + Tailwind CSS (if integrated)
- **Deployment**: Vercel (frontend hosting)
